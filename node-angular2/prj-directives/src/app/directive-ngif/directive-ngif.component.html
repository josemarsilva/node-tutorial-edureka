*ngIf - expression

*ngIf="cursos.length > 0"

Lista de cursos: {{ cursos }}

*ngIf="cursos.length == 0"

Não há cursos para serem listados

*ngIf - boolean

*ngIf="mostrarCursos"

Lista de cursos: {{ cursos }}

*ngIf="!mostrarCursos"

Não há cursos para serem listados

*ngIf - toggle mostrarCursos

hidden como alternativa ao *ngIf

[hidden]="!mostrarCursos"

[hidden]="mostrarCursos"
